# Low-tech avant les *Low-tech*

Avant le low-tech, on avait le mouvement des "technique intermédiaires" (1960-70)

>On prenait les techniques qui marchent dans les pays développés dans des pays en développement -> ça ne marchait pas. Il faut donc des techniques "*intermédiaires*"

En 1973, le choc pétrolier ramène les techniques intermédiaires dans les pays développés. (Vu que les techniques "*nouvelles*" étaient trop chères, ou pas faisable)
$\implies$ On minimalise, cherche à simplifier

Techniques *Autoritaires* vs *Démocratiques*
- Autoritaires
	- Centralisées, dépendent d'un état
	- Exemple type : centrale nucléaire -> centralisée, dangereux
- Démocratiques
	- Faibles mais durables
	- Exemple type : petite éolienne -> décentralisée, durable, environnementale
