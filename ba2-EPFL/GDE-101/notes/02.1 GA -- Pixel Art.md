# Dans Pico8

8x -> une case

très peu de couleurs !


# Palettes

-> Ensemble de couleurs.

Idéalement -> il faut limiter son utilisation des couleurs

Comment choisir ?
- Tester -> itérer sur ses choix
- Les rampes ! (Groupe de couleurs où les couleurs sont voisines)
- Ya des outils pour trouver les palettes

> [!Warning] A éviter
> - Utilisation de couleurs trop saturés
> - Couleurs trop typées (trop génériques)
> - Mettre toutes les couleurs
> - Prendre des couleurs détonantes
