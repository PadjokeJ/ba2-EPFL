Le storytelling dans le jeu vidéo se présente de manière différente 

## Game Pillar

Une ligne directrice (phrase simple) qui -- dès le début du développement -- dicte à quoi le jeu resemble

Trouver => brainstorming...
## Types d'histoire

- Une histoire de support de jeu (apporter profondeur au jeu)
- Découverte en explorant le jeu
- Histoire qui avance avec le jeu
## Raconter

Info dump => pas bien

Il faut passer par des moyens plus abstraits que juste du texte
- Cadence de jeu (eg: ressenti de stress)
- Description d'un objet (eg. dark souls légende d'un item)
- Par les mécaniques (Un joueur est **ACTIF**, pas passif)
## Univers du jeu

$\approx$ World building

Tout ce que le joueur peut intéragir avec.

Tous les jeux n'ont pas forcément besoin d'un grand monde développé.
## Personnages

Chaque personnage qui est dans le jeu devrait faire avancer l'histoire.

Le personnage principal sera joué par qqun qui n'est pas prédictible => Donner action au joueur
## Trame Narrative

Pour faire une (bonne) histoire, on peut se baser sur la trame narrative
1. Situation initiale
2. Élément perturbateur
3. Déroulement
4. Climax
5. Situation finale

On peut aussi suivre son propre schéma narratif :) on est créatifs quand même
