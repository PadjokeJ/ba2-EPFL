NAND and NOR can be used to build logic circuits

NAND + NOR is physically easier to implement and more efficient than AND+OR

NAND: $\overline{x_{1}\cdot x_{2}}$
NOR: $\overline{x_{1}+x_{2}}$

NOT can be build from both gates :
*Boolean theorems*

$\bar{x}=\overline{x\cdot x}$ --> NAND
$\bar{x}=\overline{x+x}$ --> NOR

Using DeMorgan's, we can implement any circuit from NAND (SoP)
-> This logic also applies to NOR -> (PoS)
