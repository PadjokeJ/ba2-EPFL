A circuit that selects data (multiplexer) has one or more *select* inputs, that choose which of the inputs to pass to the outputs

# 2-to-1 Multiplexer

For a three-input mutiplexer we have 
- One selection signal $s$
- Two inputs $x_{1},x_{2}$

When the signal
- $s = 0$, we have $f=x_{1}$
- $s=1$, we have $f=x_{2}$

# In general

For $n$ data inputs, we need $\lceil \log_{2}n \rceil$ select signals
