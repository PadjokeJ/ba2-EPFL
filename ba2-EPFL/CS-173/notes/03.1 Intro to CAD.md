# CAD Design Flow

*Computer Aided Design*
-> Logic

We can design using schematics (like logisim)
Or using a HDL (Hardware Description Language)

## Logic Synthesis

Translate HDL code into a network of logic gates
It derives the logic functions the circuit must realize
Then it transforms and optimizes the expressions

## Functional Simulation

A circuit can be simulated to verify if it functions as expected
Simulators assume that there is no gate delay
