The exclusive or (XOR) returns true <=> only one of the two variables is true, not both

It is written $\oplus$
$$
x_{1}\oplus x_{2}=\bar{x}_{1}x_{2}+x_{1}\bar{x}_{2}
$$
The coincidence (XNOR) returns true <=> the variables are either both true or both false

It is written $\odot$
$$
x_{1}\odot x_{2}=\bar{x}_{1}\bar{x}_{2} + x_{1}x_{2}
$$
