NMOS :
- If controlled by an input variable $x$, then the switch is open when $x=0$, else closed
PMOS:
- If controlled by an input variable $x$, then the switch is open when $x=1$

CMOS: (Equivalent to a NOT gate)
- PMOS and NMOS combined
- We have $V_{f}$, $V_{x}$, with $V_{f} = Gnd, x=1$, $V_{f} = V_{DD}, x=0$
